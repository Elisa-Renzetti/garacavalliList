<h1 id="gara-cavalli">Gara Cavalli</h1>
<p>Una simulazione di gara tra cavalli realizzata in Java usando thread per gestire l’avanzamento simultaneo di ogni cavallo.</p>
<hr>
<h2 id="project-title">1. Project title</h2>
<p><strong>Gara Cavalli</strong><br>
Una semplice applicazione Java che simula una corsa tra cavalli utilizzando il multithreading.</p>
<hr>
<h2 id="description">2. Description</h2>
<p>Il progetto <strong>Gara Cavalli</strong> simula una corsa tra più cavalli (ognuno rappresentato da un thread).<br>
Ogni cavallo percorre una distanza predefinita e aggiorna il proprio stato fino al traguardo.<br>
Alla fine, viene mostrata la <strong>classifica finale</strong> e l’utente può <strong>salvarla su file di testo</strong>.</p>
